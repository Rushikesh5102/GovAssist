# GovAssist Artifacts

This directory contains the artifacts for the GovAssist project.

## Checklist & Demo Script
See [checklist_and_demo.md](checklist_and_demo.md) for the acceptance checklist and demo video script.

## Demo Instructions
To run the demo:
1. Ensure Docker and Docker Compose are installed.
2. Run `run_demo.bat` in the root directory.
3. Access the application at `http://localhost:5173`.

## Source Code
The full source code is available in the root directory of this repository.

## Docker Image
The Docker images are built locally as `govassist-backend` and `govassist-frontend`.

# GovAssist AI: Admin Manual

**Confidential**: For Administrator Use Only.

## 1. Accessing the Admin Dashboard
1.  Log in with an admin account (e.g., `admin@govassist.in`).
2.  Navigate to `/admin` or click the "Admin Dashboard" link in the profile menu.
3.  If you are not an admin, you will be redirected to the home page.

## 2. Dashboard Overview
The dashboard provides a high-level view of the system's health:
- **Total Users**: Number of registered citizens.
- **Total Chats**: Number of active conversation sessions.
- **Active Schemes**: Number of schemes currently live in the database.

## 3. Managing Scheme Updates
The **Crawler Service** automatically finds potential new schemes. These appear in the "Pending Updates" section.

### Review Process
1.  **View Update**: Read the Title, Description, and Ministry. Click "Official Link" to verify.
2.  **Approve**: Click the **Green Checkmark**. The scheme is moved to the live database and becomes searchable.
3.  **Reject**: Click the **Red X**. The update is discarded.

### Triggering the Crawler
- Click the **"Trigger Crawler"** button in the top right.
- The system will scan configured sources and populate the pending list.
- *Note: This process may take a few minutes.*
